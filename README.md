# kalkulator waris hukum islam ✅
Kalkulator waris hukum islam berdasarkan (setara) Pokok-Pokok Hukum Waris (QS. An-Nisā' 4:7-10), Pembagian Harta Warisan (QS. An-Nisā' 4:11-14), Warisan Kalalah (QS. An-Nisā' 4:176), As-Sunnah, Kitab Faraid (Fiqih Waris), Buku II Hukum Kewarisan Kompilasi Hukum Islam.

1)	Gratis melayani (bekerja) 24/7.
2)	Tampilan 1 layar 1 halaman.
3)	Otomatis menghitung sesuai input data tanpa tombol berupa tabel pembagian harta warisan dengan keterangan (dalil) ahli waris, verifikasi.
4)	Telah diuji (audit skrip melalui gemini, copilot, claude): harta bawaan, harta bersama, harta peninggalan, tirkah, tahjiz, hutang, wasiat 1/3, al-irts, asal masalah, ta'shil, tashih, furudh, ashabah bin nafs, ashabah bil ghair, ashabah ma'al ghair, aul, radd, kalalah, gharrawain/umariyatain, musytarakah, hijab, mahjub, baitul mal.
5)	Ideal untuk alat bantu, pembanding, untuk keluarga muslim, akademisi, peneliti, notaris, advokat, KUA, PA.
6)	Anti iframe, tamper, domain lock, anti devtools, beberapa tombol keyboard, mouse klik kanan.
